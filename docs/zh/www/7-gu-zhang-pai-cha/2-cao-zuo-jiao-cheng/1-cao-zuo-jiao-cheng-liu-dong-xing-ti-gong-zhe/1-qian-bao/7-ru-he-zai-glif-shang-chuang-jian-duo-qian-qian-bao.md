# 如何在 GLIF 上创建多签钱包

使用 GLIF 多签钱包存入 FIL 代币，不仅能提升安全性，还支持多方共同管理资产。本教程将带你完成多签钱包的设置步骤。

**1. 连接到一个 Filecoin 钱包**

确保您拥有一个兼容 Filecoin 的钱包（带有 f1 钱包地址），例如 [FilSnap 钱包](/7-gu-zhang-pai-cha/2-cao-zuo-jiao-cheng/1-cao-zuo-jiao-cheng-liu-dong-xing-ti-gong-zhe/1-qian-bao/4-ru-he-li-yong-filsnap-qian-bao-zuo-wei-zhong-jie-qian-bao)、[Ledger](/7-gu-zhang-pai-cha/2-cao-zuo-jiao-cheng/1-cao-zuo-jiao-cheng-liu-dong-xing-ti-gong-zhe/1-qian-bao/2-ru-he-li-yong-ledger-qian-bao-zuo-wei-zhong-jie-qian-bao) 或 [Burner](/7-gu-zhang-pai-cha/2-cao-zuo-jiao-cheng/1-cao-zuo-jiao-cheng-liu-dong-xing-ti-gong-zhe/1-qian-bao/3-ru-he-li-yong-burner-qian-bao-zuo-wei-zhong-jie-qian-bao) 钱包。您可以点击相应的链接查看钱包的详细设置教程。

![如何在 GLIF 上创建多签钱](cn-image-304.png)

2\. **访问 GLIF 网站并进入 “多重签名钱包” 标签页**

![如何在 GLIF 上创建多签钱](cn-image-305.png)

3. **在网站左侧边栏点击“创建新的多重签名钱包**”

![如何在 GLIF 上创建多签钱](cn-image-306.png)

> [!TIP]
> 确保您的钱包中有一些 FIL 用于支付 Gas 费。

4. **添加多签签名人**

将希望纳入该多签钱包的签名人地址添加进去。如果要添加更多签名人，可点击“**添加签名者**”。

> [!IMPORTANT]
> 请注意：不支持将以太坊地址用作多签签名人，您需要使用有效的 f1 地址（例如 [FilSnap](/7-gu-zhang-pai-cha/2-cao-zuo-jiao-cheng/1-cao-zuo-jiao-cheng-liu-dong-xing-ti-gong-zhe/1-qian-bao/4-ru-he-li-yong-filsnap-qian-bao-zuo-wei-zhong-jie-qian-bao)、[Ledger](/7-gu-zhang-pai-cha/2-cao-zuo-jiao-cheng/1-cao-zuo-jiao-cheng-liu-dong-xing-ti-gong-zhe/1-qian-bao/2-ru-he-li-yong-ledger-qian-bao-zuo-wei-zhong-jie-qian-bao) 或 [Burner](/7-gu-zhang-pai-cha/2-cao-zuo-jiao-cheng/1-cao-zuo-jiao-cheng-liu-dong-xing-ti-gong-zhe/1-qian-bao/3-ru-he-li-yong-burner-qian-bao-zuo-wei-zhong-jie-qian-bao) 钱包）。

接下来，设置交易所需批准，并输入存款金额。如果需要，还可以配置vesting schedule (非必需)。确认无误后，点击“**发送**”。

![如何在 GLIF 上创建多签钱](cn-image-308.png)

5. **在钱包中确认交易**

您的钱包（如 FilSnap 或 Ledger）会弹出确认提示，点击“**确认**”即可。

![如何在 GLIF 上创建多签钱](cn-image-309.png)

在批准交易后，请耐心等待网络确认，这可能需要几分钟。

![如何在 GLIF 上创建多签钱](cn-image-310.png)

6. **访问您的多签钱包**

多签钱包创建完成后，您可以在左侧栏看到该多签钱包地址，它通常以 f2 开头。

如果没有立即看到新建的多签钱包，可尝试重新连接钱包并在“**多重签名钱包**”标签页中查看左侧列表。

在多重签名钱包页面，您可以查看该多签钱包的代币余额、提案列表、最低审批数以及交易历史等信息。

![如何在 GLIF 上创建多签钱](cn-image-311.png)

## 结论

您現在可以在 GLIF 上创建多签钱包！请务必仔细检查地址和交易详情，以确保资金安全。

若需了解如何使用多签钱包在 GLIF 池中进行存款，请参见[本教程](/7-gu-zhang-pai-cha/2-cao-zuo-jiao-cheng/1-cao-zuo-jiao-cheng-liu-dong-xing-ti-gong-zhe/2-shi-yong-glif-chi-cun-ru-he-ti-qu/3-ru-he-shi-yong-duo-qian-qian-bao-jiang-fil-cun-ru-glif)。
