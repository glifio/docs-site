# 等级与限制

GLIF+ 的等级取决于你质押的 GLF 数量，这会影响你的返现奖励和 DTL 上限 —— 等级越高，权益越多。

| 等级 | 铸造成本 ($GLF) | 质押数量 ($GLF) | 返现溢价 (Cash back premium) | DTL 上限 (杠杆) |
| ---- | --------------- | --------------- | ---------------------------- | --------------- |
| 铜卡 | 5000            | 50000           | 5%                           | 80% (5x)        |
| 银卡 | 5000            | 250000          | 10%                          | 85.7% (7x)      |
| 金卡 | 5000            | 750000          | 25%                          | 90% (10x)       |

\*DTL = 借款与清算价值比。DTL 越高，实际杠杆越大。[在此](/3-cun-chu-ti-gong-shang/3-jie-kuan-shang-xian-jie-kuan-yu-qing-suan-jia-zhi-bi-dtl)进一步了解 DTL。

### **卡片费用**

GLIF+ 卡片涉及以下三项“费用”：

1. 铸造费用 - 此为创建卡片时的一次性费用，**不会返还**给卡片持有人，也**不**包含在激活费用中。
2. 激活费用（质押要求）- 此部分为被锁定的 $GLF 代币，当卡片降级或停用后，将返还给卡片持有人。用于取得会员资格的 $GLF 是**可退还**的，并且与用于兑换返现的 $GLF 分开计算。
3. 返现计划金库 - 此部分为被锁定的 $GLF 代币，只要卡片持有人希望从返现计划中提取 $GLF，该金额即**可随时返还给卡片持有人，没有罚金。**

> [!NOTE]
> 用于利息返现的 $GLF 一旦完成兑换，将无法退还。返现兑换没有最低金额限制。

### 等级管理

卡片持有人可以根据以下规则升级或降级卡片：

#### **升级**

卡片持有人可随时免费将卡片升级到更高等级，只需质押对应等级所需的 GLF 代币。卡片必须先被激活，才能进行升级。

#### **降级**

卡片持有人可按以下规则将卡片降级到更低等级：

* 如果距离卡片的上次激活 / 升级 / 降级已超过 3 个月：
  * 可免费降级，所有用于激活的 $GLF 代币将返还给卡片持有人。
* 如果距离卡片的上次激活 / 升级 / 降级少于 3 个月：
  * 激活卡片时所质押的 $GLF 代币将被扣除 8.5% 作为罚金，持有人将收到激活时质押 $GLF 代币的 91.5%。

> [!WARNING]
> 用于卡片激活的 $GLF 代币默认锁定 3 个月。
