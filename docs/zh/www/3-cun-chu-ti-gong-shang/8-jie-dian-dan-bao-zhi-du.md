# 节点担保制度

有时 SP 控制多个独立的Miner Actor。每个Miner Actor在其抵押价值、可扩展性和地理位置方面可能有所不同。担保制度允许单个 SP 将多个Miner Actor聚合在一起，汇总所有Miner Actor的价值和借款能力。这种架构允许具有大量抵押价值的大型Miner Actor为一个小得多的Miner Actor提供抵押品以借入资金。

![节点担保制度](cn-image-377.png)

用作抵押节点的Miner Actor不需要是最终接收借款资金的Miner Actor。在 [Agents](/docs/3-cun-chu-ti-gong-shang/7-shi-yong-agents-guan-li-jie-dian) 部分裡有关于汇总借款能力的更多信息。
