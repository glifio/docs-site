# 协议目标用户

## FIL 流动性提供者

流动性提供者（Liquidity Providers，简称 LP）是将 FIL 存入 GLIF 以赚取奖励的 FIL 持有者。LP 在存入 FIL 时会收到 iFIL 代币，并通过持有 [iFIL](/2-liu-dong-xing-ti-gong-zhe/2-glif-jiang-li-ji-zhi-ifil) 代币以赚取奖励。LP 的奖励来自于从 GLIF 借入 FIL 的存储提供商所支付的费用。随着 GLIF 从存储提供商那里累积更多费用，iFIL 的价值会随着时间的推移而增加。

在[下一节](/2-liu-dong-xing-ti-gong-zhe/1-cun-ru-fil)了解更多关于成为 GLIF 流动性提供者的资讯，或查看[常见问题](../gu-zhang-pai-cha/chang-jian-wen-ti/chang-jian-wen-ti-liu-dong-xing-ti-gong-zhe/)。

您可以在我们的[网站](https://www.glif.io/zh/pool/infinity)上将 FIL 存入 GLIF，立即成为 GLIF 流动性提供者，赚取奖励。

## 存储提供商

存储提供商（Storage Providers，简称 SP）是运营一个或多个 [Filecoin Miner Acto](https://spec.filecoin.io/systems/filecoin_mining/storage_mining/)[rs](https://spec.filecoin.io/systems/filecoin_mining/storage_mining/)（“矿工”）的个人或组织。SP 提供去中心化的数据存储服务，并使用 GLIF 借入 FIL 用于存储挖矿，这有助于SP成长，同时也有助于保护整个 Filecoin 网络的安全。

作为一个 SP，你可以从大量流动性提供者 (LP) 的手中借入 FIL，而无需交出私钥、签署法律文件或进行 KYC 验证。借入的 FIL 可用于存储挖矿，或从系统中提取出来，于交易所进行交易或出售。SP 可以借入高达其抵押品 3 倍杠杆的FIL。你可以在[这一節](/3-cun-chu-ti-gong-shang/1-jie-ru-fil)了解更多关于作为存储提供者的资讯，或查看[常见问题](../gu-zhang-pai-cha/chang-jian-wen-ti/chang-jian-wen-ti-cun-chu-ti-gong-shang/)。

要立即开始从 GLIF 借入 FIL，请在我们的[网站](https://www.glif.io/zh/agent)上创建一个[Agent](/3-cun-chu-ti-gong-shang/7-shi-yong-agents-guan-li-jie-dian)。
