# GLIF Documentation Site

Source content for the [glif.io](https://glif.io) and [icn.glif.io](https://icn.glif.io) documentation sites.

## Folder structure

```
docs/
  {locale}/          # Language code: en, zh
    {domain}/        # Site domain: www, icn
      <topic folders and markdown files>
public/              # Static assets (images, PDFs) referenced from docs
```
