## **流动性提供者（LP）**

### **步骤 1：在 GLIF 网站上开始领取流程**

1. 进入 GLIF 官方网站的[空投领取页面](https://www.glif.io/en/token)。
2. 点击 “连接钱包” 以连接您符合空投资格的钱包，这个页面将显示您的 $GLF 空投额度。

![步骤 1：在 GLIF 网站上](../../.gitbook/assets/image (379).png)

3. 点击 “开始领取”，请确保您的钱包中有足够的 Gas 费来完成交易。

![步骤 1：在 GLIF 网站上](../../.gitbook/assets/image (378).png)

4. 阅读 GLIF 治理章程后，点击 “下一页”。

![步骤 1：在 GLIF 网站上](../../.gitbook/assets/image (380).png)

5. 选择您的委托人（Delegate）。$GLF 代表您的治理权，可用于 GIP 提案的投票。您可以选择 GLIF Ltd. 或其他委托人代表您投票，或选择委托给自己，之后可以随时更改设定。如果不确定应该怎么做，可以委托给 GLIF Ltd.，然后点击 “下一页”。

![Delegation option](../../.gitbook/assets/image (381).png)

6. 确认您的 $GLF 领取额度和委托选择后，点击 “领取您的代币”。

![步骤 1：在 GLIF 网站上](../../.gitbook/assets/image (382).png)

7. 在钱包中确认交易，并等待 1–2 分钟，直到交易完成。

### **步骤 2：从 NFT 兑换 $GLF 代币**

1. 点击 “返回控制面板” 或进入 “钱包” 页面。

![步骤 2：从 NFT 兑换](../../.gitbook/assets/image (383).png)

2. 向下滑到 “NFTs” 部分，您将看到一个 NFT，代表您已领取的 $GLF 代币。

{% hint style="warning" %}
空投释放时间表与 Filecoin 区块奖励同步：25% 可立即领取，75% 在 180 天内线性释放。
{% endhint %}

3. 您的 NFT 将显示可兑换的代币数量，点击 “兑换”。

![步骤 2：从 NFT 兑换](../../.gitbook/assets/image (384).png)

4. 页面将显示可兑换的数量，点击 “发送”。

![步骤 2：从 NFT 兑换](../../.gitbook/assets/image (388).png)

5. 在钱包中确认交易，并等待 1–2 分钟，直到交易完成。
6. 进入 “钱包” 页面，查看 “代币” 部分，您可以在这里查看您的 $GLF 代币！

![步骤 2：从 NFT 兑换](../../.gitbook/assets/image (389).png)

🎉 恭喜！您已成功领取 $GLF 代币，作为 GLIF 社区的早期参与者。请善用这份力量！

## **存储提供商（SP）**

### **通过 CLI 领取空投**

如果您希望使用 CLI (Command Line Interface) 领取空投，请先[安装 v2.4.0 版本](https://github.com/glifio/glif/releases/tag/v2.4.0)，并[点击这里](https://github.com/glifio/glif/blob/main/README_zh.md#airdrop-plans%E7%A9%BA%E6%8A%95%E8%AE%A1%E5%88%92)了解更多操作详情。

### **通过 GLIF 网站领取空投**

如果您希望使用 GLIF 网站界面领取，请按照以下步骤操作。

#### **步骤 1：在 GLIF 网站上开始领取流程**

1. 进入 GLIF 官方网站的[空投领取页面](https://www.glif.io/en/token)。
2. 连接到您的 Agent 拥有者地址，这个页面将显示您的 $GLF 空投额度。点击 “连接钱包”，连接您的 Agent 拥有者钱包。如果是多签钱包，请连接其中一个签名者钱包。

![步骤 1：在 GLIF 网站上](../../.gitbook/assets/image (391).png)

3. 点击 “开始领取”，请确保您的钱包中有足够的 Gas 费来完成交易。

![步骤 1：在 GLIF 网站上](../../.gitbook/assets/image (392).png)

4. 阅读 GLIF 治理章程后，点击 “下一页”。

![步骤 1：在 GLIF 网站上](../../.gitbook/assets/image (393).png)

5. 选择您的委托人（Delegate）。$GLF 代表您的治理权，可用于 GIP 提案的投票。您可以选择 GLIF Ltd. 或其他委托人代表您投票，或选择委托给自己，之后可以随时更改设定。如果不确定应该怎么做，可以委托给 GLIF Ltd.，然后点击 “下一页”。

![Delegation option](../../.gitbook/assets/image (394).png)

6. 确认您的 $GLF 领取额度和委托选择后，点击 “领取您的代币”。

![步骤 1：在 GLIF 网站上](../../.gitbook/assets/image (395).png)

7. 在钱包中确认交易，并等待 1–2 分钟，直到交易完成。

#### **步骤 2：批准领取及委托的多签提案**

1. 点击 “返回控制面板” 或进入 “多重签名钱包” 页面。

_如果您的 Agent 拥有者是**单签/EOA（外部拥有账户）钱包**，则无需签署任何提案，您可以直接跳到_[_第 3 步_](ru-he-ling-qu-glf-kong-tou.md#bu-zhou-3-cong-nft-dui-huan-glf-dai-bi)_。_

![步骤 2：批准领取及委托的多签](../../.gitbook/assets/image (396).png)

2. 点击 “管理员”，然后找到 “多重签名钱包提案” 部分，您会看到一个 “Claim And Delegate” 提案。

![步骤 2：批准领取及委托的多签](../../.gitbook/assets/image (398).png)

3. 断开当前钱包，并连接到您的 Agent 拥有者多签钱包中的其他签名者钱包。
4. 在 “多重签名钱包” 页面中，找到 “多重签名钱包提案” 部分，在同一个 “Claim And Delegate” 提案旁边点击 ✅ 符号。

![步骤 2：批准领取及委托的多签](../../.gitbook/assets/image (399).png)

5. 点击 “批准提案”。

![步骤 2：批准领取及委托的多签](../../.gitbook/assets/image (400).png)

6. 在钱包中确认交易，并等待 1–2 分钟，直到交易完成。

#### **步骤 3：从 NFT 兑换 $GLF 代币**

1. 所有批准人批准提案后，点击 “资产组合” 并下滑至 “NFT” 部分，您将看到一个 NFT，代表您已领取的 $GLF 代币。

_如果您的 Agent 拥有者是单签钱包或 EOA（外部拥有账户），则无需签署任何提案。只需点击 NFT 上的“兑换”按钮，并在钱包中确认交易。交易完成后，查看您的资产组合以获取 $GLF 代币。您的 NFT 余额会自动更新，如果NFT还有尚未兑换的代币_，_之后您可以随时再次兑换代币。_

{% hint style="warning" %}
空投释放时间表与 Filecoin 区块奖励同步：25% 可立即领取，75% 在 180 天内线性释放。
{% endhint %}

2. 点击 “兑换” 以发起另一个多签提案。

![步骤 3：从 NFT 兑换](../../.gitbook/assets/image (401).png)

2. 点击 “发送”，此操作将创建另一个多签提案来兑换 $GLF 代币。在钱包中确认交易。

![步骤 3：从 NFT 兑换](../../.gitbook/assets/image (402).png)

3. 交易完成后，进入 “管理员” 部分，您将看到一个新的 “Redeem Plans” 提案。

![步骤 3：从 NFT 兑换](../../.gitbook/assets/image (405).png)

4. 断开当前钱包，并连接到 Agent 所有者多签钱包中的其他签名者钱包。
5. 在 “多重签名钱包” 页面中，找到 “多重签名钱包提案” 部分中的同一个 “Redeem Plans” 提案，并点击 “批准提案”。

![步骤 3：从 NFT 兑换](../../.gitbook/assets/image (404).png)

6. 在钱包中确认交易。
7. 交易完成后，在资产组合检查您的 $GLF 代币，您的 NFT 余额将会更新，如果NFT还有尚未兑换的代币，您可以随时继续兑换代币。

![步骤 3：从 NFT 兑换](../../.gitbook/assets/image (403).png)

🎉 恭喜！您已成功领取 $GLF 代币，作为 GLIF 社区的早期参与者。请善用这份力量！
